# Aide Google Images dans Modulys Magic Lock

Ce patch ajoute directement dans l'application des boutons d'aide pour récupérer les deux informations nécessaires au mode Google Images :

- **Google API Key**
- **Google Search Engine ID / CX**

Liens ajoutés dans l'application :

1. Activer l'API Custom Search JSON  
   https://console.cloud.google.com/apis/library/customsearch.googleapis.com

2. Créer ou copier une clé API  
   https://console.cloud.google.com/apis/credentials

3. Créer un moteur Programmable Search  
   https://programmablesearchengine.google.com/controlpanel/create

4. Retrouver son Search Engine ID / CX  
   https://programmablesearchengine.google.com/controlpanel/all

Réglage conseillé : dans Programmable Search Engine, activer la recherche sur tout le Web pour pouvoir chercher des images générales : cartes, objets, célébrités, logos, etc.
